KeyLume Listing Manager

A private seller tool for managing digital artwork listings on Etsy

About This Tool

KeyLume Listing Manager is a private, single-shop tool built for KeyLumeStudio on Etsy. It helps streamline the process of preparing and managing listings for digital artwork and downloadable art files.

This tool assists with drafting listings, organizing product images, applying correct taxonomy and tags, and keeping inventory data in sync. All listings are reviewed and published manually by the shop owner before going live.

What It Does

📝

Draft Preparation

Helps prepare listing drafts with accurate titles, descriptions, tags, and taxonomy categories.

🖼️

Image Management

Organizes and manages product photography and mockup images for listings.

📦

Inventory Sync

Keeps local inventory records in sync with the shop's listing data on Etsy.

🏷️

Taxonomy & Tags

Applies correct Etsy categories and optimized tags to improve listing discoverability.

API Scopes Requested

This application requests only the minimum permissions necessary to function:

listings_r
listings_w
listings_d
shops_r

How Data Is Handled

  • Used exclusively by one shop owner for a single Etsy store (KeyLumeStudio)
  • No user data is collected, stored, or shared with any third parties
  • OAuth tokens are stored securely and used only for authenticated API requests
  • No customer or buyer data is accessed or processed at any time
  • The tool runs locally and does not operate as a public-facing service

Privacy Policy
